Sort Algorithms

Posted on 2021-08-09 In Algorithm Views: Valine:

本文总结了各种基础的排序算法，包括选择排序，插入排序，希尔排序，归并排序，快速排序，优先队列（堆排序）等等。每个算法都归结了其基本思想，实现代码和算法分析。

1. 选择排序

1.1 基本思想

(1) 首先，找到数组中最小的那个元素。

(2) 其次，将它和数组的第一个元素交换位置。(如果第一个元素就是最小元素那么他就和自己交换。)

(3)再次，在剩下的数组中找到最小的元素，将它与数组的第二个元素交换位置。

(4) 重复上述过程，直到将数组排序。

1.2 代码

1.3 分析

选择排序有两个鲜明的特点：

运行时间与输入无关。一个已经有序的数组和一个元素随机排列的数组，排序所需时间一样。

数据移动是最少的。整个过程只用了$N$次交换。

对于长度为 $N$ 的数组，选择排序需要大约 $N^2/2$ 次比较和 $N$ 次交换。

2. 插入排序

2.1 基本思想

从第二个元素开始，将每个元素与前面已排序的子数组比较，并将它插入到合适的位置。与选择排序一样，当前索引左边的所有元素都是有序的，但它们的最终位置还不确定，为了给更小的元素腾位置，它们可能会被移动。

2.2 代码

2.3 分析

对一个有序数组进行排序时，插入排序能够立即发现每个元素都已经在合适的位置上，它的运行时间也是线性的（对于这种数组，选择排序的运行时间是平方级别的）。对于所有主键相同的数组也会出现相同的情况。

3. 希尔排序

3.1 基本思想

希尔排序的思想是使数组中任意间隔为h的元素都是有序的。即将h个子数组进行插入排序。h从一个较大的数递减至1。

4.归并排序

4.1 基本思想

4.1.1 自顶向下的归并排序

对于子数组 a[lo..hi], 先将它分为 a[lo..mid] 和 a[mid+1..hi] 两部分，分别通过递归调用将它们单独排序，最后将有序的子数组归并为最终的排序结果。

4.1.2 自底向上的归并排序

实际上我们归并的数组大多数非常小。因此，首先我们进行两两归并，然后四四归并，然后是八八的归并，一直下去。

例如对于N=8的数组，我们先分别归并a[0]a[1], a[2]a[3], a[4]a[5], a[6]a[7], 然后分别归并 a[0],a[1],a[2],a[3] 和 a[4],a[5],a[6],a[7]，最后归并a[0]~a[7]。
