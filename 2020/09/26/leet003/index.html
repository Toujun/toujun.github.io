Leetcode 03:Longest Substring Without Repeating Characters

Posted on 2020-09-26 In Leetcode Views: Valine:

描述：给定字符串 s，输出字符串的最长无重复子字符串的长度。

难度：Medium

思路：

遍历字符串。

  1. 如果 hashMap 中不含有当前字符，将当前字符和所在的位置 ( i+1 ) 以键值对的方式加入 hashMap。
  2. 如果 hashMap 中含有当前字符，说明该字符重复了。此时获取该字符上次出现时的位置，赋给一个新的参数 j。同时更新 hashMap，将当前字符的所在位置更新为现在的位置。
  3. 此时用当前字符的当前位置 i+1 减去之前出现的位置 j，就可以得到子字符串的长度。
  4. 但是此时得到的字符串可能并不是最长子字符串。因此每次碰到一个重复的字符，都要计算由它引起的无重复子字符串的长度，并和之前得到的无重复子字符串长度进行比较，保存较大的长度。
  5. 遍历结束输出保存的最大长度即可。
