Mac Idea下的 JavaWeb 项目打包部署到云服务器

Posted on 2020-09-26 In Java Views: Valine:

环境一览

MacOS, IDEA, JDK, Mysql, Tomcat, Viper FTP, 腾讯云

部署步骤

  1. 腾讯云购买云服务器实例，并购买域名，将二者关联起来。购买完域名需尽快实名认证，否则不能使用。
  2. 登录腾讯云，初始密码在站内信中通知。
  3. 安装 JDK 并配置环境变量。然后安装 Mysql 和 Tomcat。这三项都可以到官网下载再利用 FTP 软件上传到云服务器，也可以都使用命令行下载。
  4. 下载完毕测试这三项是否成功安装。JDK 和 Mysql 在终端查看版本即可。Tomcat 需先 start 然后通过云服务器实例的公网地址加 8080 端口来访问。都没问题了进行下一步。
  5. 如果你的工程和数据库使用的是同一个服务器，那么工程里面连接数据库的 url 可以直接写 localhost:8080，否则需要将 localhost 替换成数据库所在服务器的公网 ip。
  6. 进入 Idea 工程，进入 File - Project Structure - Artifacts, 点击加号创建 Web Application Archive - for “your project name”，保存到你想要的文件夹中。然后点击 Build - Build Artifacts, 选择刚刚创建的 war ，action 点击build。
  7. 将生成的 war 包通过 FTP 软件上传到云服务器的 tomcat/webapps 下。这个时候通过公网 ip:8080/yourProjectName 即可访问该工程。域名绑定完成后即可用域名替换公网 ip 进行访问。
  8. 当然我们实际使用的时候肯定希望省略 :8080 和工程名，这个时候需要到 tomcat 下的配置文件里面进行简单的配置即可。
  9. 如果想在本地利用数据库图形化软件（例如 Workbench）访问云服务器上的数据库，则首先要在云服务器上将数据库开通远程访问权限，然后在 workbench 里面创建新的连接，输入公网 ip 和密码即可访问。

以上。
