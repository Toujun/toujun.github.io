Leetcode 15:3Sum

Posted on 2020-09-29 In Leetcode Views: Valine:

描述：给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。注意：答案中不可以包含重复的三元组。

难度：Medium

思路：

  1. 首先将数组进行排序，默认为从小到大的顺序。并声明一个类型为整型List的集合set用来存放结果。
  2. 声明左指针指向当前位置+1，右指针指向数组最后一位。
  3. 判断当前位置的数和两个指针指向的数相加是否为 0。如果和大于 0 ，由于数组排序后按照从小到大排列了，所以将右指针左移，相当于加上一个较小的数，有可能得到 0。如果和小于0，则将左指针右移，相当于加上一个较大的数，有可能得到 0。
  4. 如果当前三个数的和正好等于 0，则将这三个数字加入到结果集 set 中。然后同时将左指针右移及右指针左移，因为这样也可能得到 0。
