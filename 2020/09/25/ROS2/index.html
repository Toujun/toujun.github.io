Android 的 ROS 开发环境搭建(Android+ROS+rosjava+android_core)

Posted on 2020-09-25 In Ros Views: Valine:

下载安装并配置Android Studio

1.首先下载安装 Android Studio 和 android sdk。

点我下载

2.ROS 系统自带的是 openjdk。当然，Android开发需要oracle的jdk。

安装jdk8并配置

3.对于64位的系统，需要安装ia-32的包。

4.将Android Studio放到/opt目录下，并配置环境变量。

5.启动Android SDK Manager。

安装Android API 19 / 18 / 17 / 16 /15 / 13 / 10 / 7。为了避免sdk占用太多磁盘空间，可取消勾选system image选项。

6.启动Android Studio。

给Android Studio配置ros环境

7.安装相关的包。

8.安装Wstool工具。

10.下载android_core并source一下，让该文件夹和rosjava相关联。
