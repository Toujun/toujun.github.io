javaweb阶段学习总结

Posted on 2020-09-25 In Java Views: Valine:

本文是看完尚硅谷王振国老师Javaweb视频后的技术总结。

HTML 部分

  1. 超链接的 target 属性来设置当前页面跳转或跳转到新页面 _self / _blank 超链接去下划线：text-decoration:none;
  2. 使用 colspan 和 rowspan来设置跨列和跨行
  3. iframe 标签可以设置内嵌窗口
  4. 表单里可以设置隐藏域 input hidden
  5. 标签名选择器，id选择器 #id，class选择器 .class，组合选择器

后端部分

  1. JavaWeb 三大组件：Servlet，Filter，Listener

  2. web.xml 中需配置servlet程序的访问地址 <servlet>及<servlet-mapping>

  3. Servlet 的生命周期：

    • 构造器方法 （第一次访问时调用）

    • init初始化方法（第一次访问时调用）

    • service方法（每次访问都会调用）

    • destroy方法 （在web工程停止时调用）

  4. 常见 HTTP 响应码

    200 OK，302 重定向，404 数据不存在，500 服务器内部错误

  5. JavaEE 三层架构：Web层，Service业务层，Dao持久层

  6. Jsp 主要作用是代替Servlet程序回传 html 页面的数据，本质上是一个Servlet程序。

  7. Jsp 四大域对象：pageContext，request，session，application。域对象是可以像Map一样存取数据的对象。

  8. EL 表达式：代替 jsp 页面中的表达式脚本在 jsp 页面中进行数据的输出。通过pageScope, requestScope, sessionScope, applicationScope 来获取四个特定域中的属性。

  9. JSTL 标签库：代替 jsp 页面中的代码脚本。

    • 导入 jar 包，在 jsp 页面使用 taglib 指令引入标签库
    • 常用的 Core 标签库，prefix=”c”

  10. 文件的上传步骤

    • 要有一个 form 标签，method=post
    • form 标签的 encType 属性值必须为 multipart/form-data
    • 在 form 标签中使用 input type=file 添加上传的文件
    • 编写服务器代码（Servlet）接收，处理上传的数据

  11. 页面 jsp 动态化

    • 在 html 页面顶行添加 page 指令
    • 修改文件后缀名为 .jsp
    • 搜索替换页面中的 .html 为 .jsp

  12. Cookie

    • 客户端保存键值对，每个Cookie大小不超过 4kb
    • setMaxAge()，正数表示在指定的秒数后过期；负数表示浏览器一关就被删除（默认-1）；零表示马上删除

  13. Session

    • 通常用来保存用户登录过后的信息

    • Session 默认超时时长为30分钟，可以在 web.xml 中通过 session-config 来配置默认时长

  14. 验证码的实现步骤

    • 导入谷歌验证码 jar 包 kaptcha
    • 在 web.xml 中配置用于生成验证码的 Servlet 程序
    • 在表单中使用 img 标签去显示验证码并使用 <img src=”/kaptcha.jpg”>
    • 在服务器获取生成的验证码和客户端发过来的验证码比较
    • 给验证码绑定单击事件，使客户端可以切换验证码

  15. Filter 作用：拦截请求，过滤响应。常用于权限检查，日记操作，事务管理等等。

    • 编写一个类去实现 Filter 接口
    • 实现过滤方法 doFilter()
    • 到 web.xml 中配置Filter的拦截路径

  16. Filter 的生命周期

    • 构造器方法 （工程启动时执行）
    • init 初始化方法（工程启动时执行）
    • doFilter 过滤方法，每次拦截到请求就执行
    • destroy 方法，工程停止时执行

  17. ThreadLocal 简介

    • ThreadLocal 可以为当前线程关联一个数据。(它可以像 Map 一样存取数据，key 为当前线程)
    • 每一个 ThreadLocal 对象，只能为当前线程关联一个数据，如果要为当前线程关联多个数据，就需要使用多个 ThreadLocal 对象实例。
    • 每个 ThreadLocal 对象实例定义的时候，一般都是 static 类
    • ThreadLocal 中保存数据，在线程销毁后。会由 JVM 虚拟自动释放。

  18. 使用 Filter 和 ThreadLocal 组合管理事务

    • 目的是确保所有 dao 操作都在同一个 Connection 连接对象中完成
    • 使用 Filter 统一给所有的 Service 方法加上 try-catch 来实现事务管理

  19. 在 web.xml 中可以使用 <error-page>来设置错误信息页面

  20. jQuery 中的 AJAX 请求

    • $.ajax 方法：url, type, data, success, dataType
    • $.getJSON 方法：url, data, callback

  21. i18n 国际化

    • 三要素：Locale对象，ResourceBundle资源包，Properties属性配置文件
    • 使用步骤：从请求头中获取 Locale 信息；根据指定的baseName 和 Locale对象，用 ResourceBundle 读取包
